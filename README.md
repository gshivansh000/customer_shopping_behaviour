🛍️ Customer Shopping Behavior Analysis

This project analyzes customer shopping patterns using Python, SQL, and Power BI.
The goal is to understand purchasing behavior, identify key revenue drivers, and provide business insights that help improve marketing and customer engagement strategies.

📌 Project Overview

The analysis is based on 3,900 customer purchase transactions containing demographic and behavioral information.

Key objectives of the project:

Analyze customer purchasing patterns

Identify high-value customers

Understand the impact of discounts and subscriptions

Discover popular product categories

Provide data-driven business recommendations

📊 Dataset

Dataset features include:

Customer ID

Age and Gender

Location

Item Purchased

Product Category

Purchase Amount

Review Rating

Subscription Status

Shipping Type

Discount Applied

Previous Purchases

⚙️ Project Workflow
1️⃣ Data Loading

Dataset imported using Python and Pandas.

2️⃣ Data Cleaning

Handled missing values

Standardized column names

Created new features such as age groups

3️⃣ Exploratory Data Analysis (EDA)

Performed statistical analysis and visualizations to understand trends.

4️⃣ SQL Analysis

Used SQL queries to answer business questions such as:

Revenue by gender

Customer segmentation

Top-rated products

Subscription impact

5️⃣ Power BI Dashboard

Built an interactive dashboard to visualize insights including:

Revenue by category

Sales by age group

Subscription distribution

Average purchase amount

📈 Key Insights

Express shipping customers spend more per transaction

Subscription customers contribute a significant share of revenue

Some products are highly dependent on discounts

Loyal customers represent the largest customer segment
